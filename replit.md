# Резюме.Про | ResumePro AI — VK Bot

VK-бот для группы rezume_pro, который помогает пользователям улучшать резюме.

## Структура проекта

- `main.py` — основной файл бота (Flask + VK Bot LongPoll)
- `requirements.txt` — зависимости Python

## Зависимости

- `vk_api==11.10.0` — работа с VK API
- `python-dotenv==1.2.1` — загрузка переменных окружения
- `requests==2.32.5` — HTTP-запросы
- `Flask==3.0.0` — веб-сервер для поддержания активности

## Конфигурация

Необходимые переменные окружения (Secrets):
- `VK_TOKEN` — токен бота ВКонтакте (из настроек группы → Работа с API)
- `GROUP_ID` — ID группы ВКонтакте (по умолчанию: 237022345)

## Запуск

Workflow "Start application" запускает `python main.py`:
- Flask сервер на порту 5000 (веб-превью и keepalive)
- VK LongPoll бот запускается в основном потоке

## Команды бота

- `/start`, `привет`, `начать` — приветственное сообщение
- `/demo`, `демо`, `пример` — пример улучшения резюме
